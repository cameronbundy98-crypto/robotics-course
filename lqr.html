Setup

When Does LQR Apply?

LQR solves optimal control problems where

  • time is discrete, t∈{0,1,2,…,T}t \in \{0, 1, 2, \dots, T\}
  • the dynamics are linear (or affine)
  • the cost is quadratic in state and control

These slides follow Sergey Levine’s formulation, which generalizes naturally to iLQR and connects to reinforcement learning.

System and Cost

At each time tt, the dynamics and cost are:

𝐱t+1=At[𝐱t𝐮t]+𝐚t\mathbf{x}_{t+1} = A_t \begin{bmatrix}\mathbf{x}_t \\ \mathbf{u}_t\end{bmatrix} + \mathbf{a}_t

ct(𝐱t,𝐮t)=12[𝐱t𝐮t]T𝐂t[𝐱t𝐮t]+[𝐱t𝐮t]T𝐜tc_t(\mathbf{x}_t, \mathbf{u}_t) = \frac{1}{2}\begin{bmatrix}\mathbf{x}_t \\ \mathbf{u}_t\end{bmatrix}^{\!T} \mathbf{C}_t \begin{bmatrix}\mathbf{x}_t \\ \mathbf{u}_t\end{bmatrix} + \begin{bmatrix}\mathbf{x}_t \\ \mathbf{u}_t\end{bmatrix}^{\!T} \mathbf{c}_t

Total cost over the finite horizon:

J=∑t=0Tct(𝐱t,𝐮t)J = \sum_{t=0}^{T} c_t(\mathbf{x}_t, \mathbf{u}_t)

The terminal cost cT(𝐱T,𝐮T)c_T(\mathbf{x}_T, \mathbf{u}_T) is included in the sum. A typical choice is cT=12𝐱TT𝐂𝐱𝐱,T𝐱Tc_T = \frac{1}{2}\mathbf{x}_T^T \mathbf{C}_{\mathbf{x}\mathbf{x},T} \mathbf{x}_T (no control at final step).

Double Integrator: Applying the Notation

A unit-mass particle: p̈=u\ddot{p} = u. State 𝐱t=[pt,vt]T\mathbf{x}_t = [p_t,\, v_t]^T, control 𝐮t=[at]\mathbf{u}_t = [a_t].

Discrete dynamics (timestep Δt\Delta t):

At=[1ΔtΔt2201Δt],𝐚t=𝟎A_t = \begin{bmatrix}1 & \Delta t & \tfrac{\Delta t^2}{2} \\ 0 & 1 & \Delta t\end{bmatrix}, \quad \mathbf{a}_t = \mathbf{0}

Quadratic cost with no cross terms and 𝐜t=𝟎\mathbf{c}_t = \mathbf{0}:

𝐂t=[qp000qv000r],𝐂T=[qf00qf]\mathbf{C}_t = \begin{bmatrix} q_p & 0 & 0 \\ 0 & q_v & 0 \\ 0 & 0 & r \end{bmatrix}, \quad \mathbf{C}_T = \begin{bmatrix} q_f & 0 \\ 0 & q_f \end{bmatrix}

AtA_t maps the augmented state [𝐱t;𝐮t][\mathbf{x}_t;\, \mathbf{u}_t] to 𝐱t+1\mathbf{x}_{t+1}. The columns correspond to [pt,vt,at][p_t,\, v_t,\, a_t].

Dynamic Programming

The Q-Function

Define the action-value function (Q-function) at time tt:

𝐐t(𝐱t,𝐮t)=ct(𝐱t,𝐮t)+Vt+1*(At[𝐱t𝐮t]+𝐚t)\mathbf{Q}_t(\mathbf{x}_t, \mathbf{u}_t) = c_t(\mathbf{x}_t,\mathbf{u}_t) + V_{t+1}^*\!\left(A_t\begin{bmatrix}\mathbf{x}_t\\\mathbf{u}_t\end{bmatrix} + \mathbf{a}_t\right)

Since Vt+1*(𝐱)=12𝐱T𝐕t+1𝐱+𝐱T𝐯t+1V_{t+1}^*(\mathbf{x}) = \frac{1}{2}\mathbf{x}^T \mathbf{V}_{t+1} \mathbf{x} + \mathbf{x}^T \mathbf{v}_{t+1} is quadratic, 𝐐t\mathbf{Q}_t is also quadratic:

𝐐t(𝐱t,𝐮t)=const+12[𝐱t𝐮t]T𝐐̂t[𝐱t𝐮t]+[𝐱t𝐮t]T𝐪̂t\mathbf{Q}_t(\mathbf{x}_t,\mathbf{u}_t) = \mathrm{const} + \frac{1}{2}\begin{bmatrix}\mathbf{x}_t\\\mathbf{u}_t\end{bmatrix}^{\!T}\hat{\mathbf{Q}}_t\begin{bmatrix}\mathbf{x}_t\\\mathbf{u}_t\end{bmatrix} + \begin{bmatrix}\mathbf{x}_t\\\mathbf{u}_t\end{bmatrix}^{\!T}\hat{\mathbf{q}}_t

where

𝐐̂t=𝐂t+AtT𝐕t+1At,𝐪̂t=𝐜t+AtT𝐕t+1𝐚t+AtT𝐯t+1\hat{\mathbf{Q}}_t = \mathbf{C}_t + A_t^T \mathbf{V}_{t+1} A_t, \qquad \hat{\mathbf{q}}_t = \mathbf{c}_t + A_t^T \mathbf{V}_{t+1} \mathbf{a}_t + A_t^T \mathbf{v}_{t+1}

Optimal Control at Each Step

Minimize 𝐐t\mathbf{Q}_t over 𝐮t\mathbf{u}_t. Partition the Q-function matrix:

𝐐̂t=[𝐐̂𝐱𝐱𝐐̂𝐱𝐮𝐐̂𝐮𝐱𝐐̂𝐮𝐮],𝐪̂t=[𝐪̂𝐱𝐪̂𝐮]\hat{\mathbf{Q}}_t = \begin{bmatrix} \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{x}} & \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{u}} \\ \hat{\mathbf{Q}}_{\mathbf{u}\mathbf{x}} & \hat{\mathbf{Q}}_{\mathbf{u}\mathbf{u}} \end{bmatrix}, \qquad \hat{\mathbf{q}}_t = \begin{bmatrix} \hat{\mathbf{q}}_{\mathbf{x}} \\ \hat{\mathbf{q}}_{\mathbf{u}} \end{bmatrix}

Setting ∇𝐮t𝐐t=0\nabla_{\mathbf{u}_t} \mathbf{Q}_t = 0:

𝐮t*=𝐊t𝐱t+𝐤t\boxed{\mathbf{u}_t^* = \mathbf{K}_t \mathbf{x}_t + \mathbf{k}_t}

𝐊t=−𝐐̂𝐮𝐮−1𝐐̂𝐮𝐱,𝐤t=−𝐐̂𝐮𝐮−1𝐪̂𝐮\mathbf{K}_t = -\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{u}}^{-1}\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{x}}, \qquad \mathbf{k}_t = -\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{u}}^{-1}\hat{\mathbf{q}}_{\mathbf{u}}

The optimal policy is affine in state: 𝐮t*=𝐊t𝐱t+𝐤t\mathbf{u}_t^* = \mathbf{K}_t \mathbf{x}_t + \mathbf{k}_t. When 𝐜t=0\mathbf{c}_t = 0 and 𝐚t=0\mathbf{a}_t = 0, then 𝐤t=0\mathbf{k}_t = 0 (linear policy).

Value Function Update

Substituting 𝐮t*\mathbf{u}_t^* back into 𝐐t\mathbf{Q}_t gives the value function:

Vt*(𝐱t)=12𝐱tT𝐕t𝐱t+𝐱tT𝐯t+constV_t^*(\mathbf{x}_t) = \frac{1}{2}\mathbf{x}_t^T \mathbf{V}_t \mathbf{x}_t + \mathbf{x}_t^T \mathbf{v}_t + \mathrm{const}

where

𝐕t=𝐐̂𝐱𝐱+𝐐̂𝐱𝐮𝐊t\mathbf{V}_t = \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{x}} + \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{u}}\mathbf{K}_t

𝐯t=𝐪̂𝐱+𝐐̂𝐱𝐮𝐤t\mathbf{v}_t = \hat{\mathbf{q}}_{\mathbf{x}} + \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{u}}\mathbf{k}_t

This is again quadratic — so the structure repeats at t−1,t−2,…,0t-1, t-2, \dots, 0.

Algorithm

Finite Horizon LQR

Inputs: AtA_t, 𝐚t\mathbf{a}_t, 𝐂t\mathbf{C}_t, 𝐜t\mathbf{c}_t for t=0…Tt=0\dots T

Backward pass — initialize 𝐕T=𝐂𝐱𝐱,T\mathbf{V}_T = \mathbf{C}_{\mathbf{x}\mathbf{x},T}, 𝐯T=𝐜𝐱,T\mathbf{v}_T = \mathbf{c}_{\mathbf{x},T}; for t=T−1,…,0t = T-1,\dots,0:

  1. 𝐐̂t=𝐂t+AtT𝐕t+1At\hat{\mathbf{Q}}_t = \mathbf{C}_t + A_t^T \mathbf{V}_{t+1} A_t
  2. 𝐪̂t=𝐜t+AtT(𝐕t+1𝐚t+𝐯t+1)\hat{\mathbf{q}}_t = \mathbf{c}_t + A_t^T(\mathbf{V}_{t+1}\mathbf{a}_t + \mathbf{v}_{t+1})
  3. 𝐊t=−𝐐̂𝐮𝐮−1𝐐̂𝐮𝐱\mathbf{K}_t = -\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{u}}^{-1}\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{x}}, 𝐤t=−𝐐̂𝐮𝐮−1𝐪̂𝐮\quad\mathbf{k}_t = -\hat{\mathbf{Q}}_{\mathbf{u}\mathbf{u}}^{-1}\hat{\mathbf{q}}_{\mathbf{u}}
  4. 𝐕t=𝐐̂𝐱𝐱+𝐐̂𝐱𝐮𝐊t\mathbf{V}_t = \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{x}} + \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{u}}\mathbf{K}_t, 𝐯t=𝐪̂𝐱+𝐐̂𝐱𝐮𝐤t\quad\mathbf{v}_t = \hat{\mathbf{q}}_{\mathbf{x}} + \hat{\mathbf{Q}}_{\mathbf{x}\mathbf{u}}\mathbf{k}_t

Forward pass — given 𝐱0\mathbf{x}_0, for t=0,…,T−1t = 0,\dots,T-1:

  1. 𝐮t*=𝐊t𝐱t+𝐤t\mathbf{u}_t^* = \mathbf{K}_t\mathbf{x}_t + \mathbf{k}_t
  2. 𝐱t+1=At[𝐱t;𝐮t*]+𝐚t\mathbf{x}_{t+1} = A_t[\mathbf{x}_t;\,\mathbf{u}_t^*] + \mathbf{a}_t

def server(input, output, session):
    @render.plot
    def lqr_plot():
        import matplotlib.pyplot as plt
        import matplotlib.gridspec as gridspec

        dt = 0.1
        T  = input.T()
        qp, qv, r, qf = input.qp(), input.qv(), input.r(), input.qf()
        x0 = np.array([input.p0(), input.v0()])

        # Augmented dynamics: x_{t+1} = A_aug @ [x_t; u_t]
        A_aug = np.array([[1.0, dt, 0.5*dt**2],
                          [0.0, 1.0, dt]])

        # Cost matrix C_t (3x3 augmented), c_t = 0
        C = np.diag([qp, qv, r])

        # Terminal value function V_T, v_T
        V = np.diag([qf, qf])
        v = np.zeros(2)

        # --- Backward pass ---
        Ks, ks = [], []
        for _ in range(T):
            Q_hat = C + A_aug.T @ V @ A_aug          # 3x3
            q_hat = A_aug.T @ v                       # 3  (a_t=0, c_t=0)
            Qxx, Qxu = Q_hat[:2, :2], Q_hat[:2, 2:]
            Qux, Quu = Q_hat[2:, :2], Q_hat[2:, 2:]
            qx,  qu  = q_hat[:2],     q_hat[2:]
            Quu_inv  = np.linalg.inv(Quu)
            K = -Quu_inv @ Qux                        # 1x2
            k = -Quu_inv @ qu                         # 1
            V = Qxx + Qxu @ K
            v = qx  + Qxu @ k
            Ks.append(K); ks.append(k)
        Ks.reverse(); ks.reverse()

        # --- Forward pass ---
        xs = np.zeros((T + 1, 2))
        us = np.zeros(T)
        Js = np.zeros(T+1)
        xs[0] = x0
        J = 0.0
        for t in range(T):
            u = (Ks[t] @ xs[t] + ks[t]).flatten()
            us[t] = u[0]
            z = np.concatenate([xs[t], u])
            xs[t+1] = A_aug @ z
            J += 0.5*(z @ C @ z)
            Js[t] = J
        J += 0.5*(xs[T] @ np.diag([qf, qf]) @ xs[T])
        Js[T]=J

        # --- Plot ---
        time = np.arange(T + 1) * dt
        fig  = plt.figure(figsize=(8, 6))
        gs   = gridspec.GridSpec(3, 1, hspace=0.55)

        ax1 = fig.add_subplot(gs[0])
        ax1.plot(time, Js, 'b-o', markersize=3, lw=1.5)
        ax1.axhline(0, color='k', lw=0.8, ls='--')
        ax1.set_ylabel("Cost $J$")
        ax1.set_title(f"Optimal Trajectory  (J = {J:.3f})", fontsize=10)
        ax1.grid(True, alpha=0.3)

        ax2 = fig.add_subplot(gs[1])
        ax2.plot(xs[:,0], xs[:, 1], 'g-o', markersize=3, lw=1.5)
        ax2.axhline(0, color='k', lw=0.8, ls='--')
        ax2.set_xlabel("Position $p$")
        ax2.set_ylabel("Velocity $v$")
        ax2.grid(True, alpha=0.3)

        ax3 = fig.add_subplot(gs[2])
        ax3.step(time[:-1], us, 'r-', where='post', lw=1.5)
        ax3.axhline(0, color='k', lw=0.8, ls='--')
        ax3.set_ylabel("Control $u$")
        ax3.set_xlabel("Time (s)")
        ax3.grid(True, alpha=0.3)

        fig.tight_layout()
        return fig

app = App(app_ui, server)

Summary

Key Points

Symbol Meaning
AtA_t, 𝐚t\mathbf{a}_t Augmented dynamics (maps [𝐱t;𝐮t][\mathbf{x}_t;\mathbf{u}_t] to 𝐱t+1\mathbf{x}_{t+1})
𝐂t\mathbf{C}_t, 𝐜t\mathbf{c}_t Quadratic and linear cost parameters
𝐐t(𝐱t,𝐮t)\mathbf{Q}_t(\mathbf{x}_t,\mathbf{u}_t) Q-function (action-value function)
𝐐̂t\hat{\mathbf{Q}}_t, 𝐪̂t\hat{\mathbf{q}}_t Matrix/vector of Q-function
𝐕t\mathbf{V}_t, 𝐯t\mathbf{v}_t Matrix/vector of value function Vt*V_t^*
𝐊t\mathbf{K}_t, 𝐤t\mathbf{k}_t Feedback gain and feedforward term

Connection to RL: 𝐐t\mathbf{Q}_t is the Q-function; Vt*V_t^* is the value function. The backward pass is exactly policy evaluation + improvement for LQ systems.
