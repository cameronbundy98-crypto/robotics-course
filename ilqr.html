Recap

The Setup

Nominal trajectory x‾0,u‾0,…,x‾T−1,u‾T−1,x‾T,u‾T−1,x‾T\bar{x}_0, \bar{u}_0, \dots, \bar{x}_{T-1}, \bar{u}_{T-1}, \bar{x}_{T}, \bar{u}_{T-1}, \bar{x}_{T} satisfies the true nonlinear dynamics.

Define deviations: δxt=xt−x‾t\delta x_{t} = x_{t} - \bar{x}_{t}, δut=ut−u‾t\quad \delta u_{t} = u_{t} - \bar{u}_{t}.

Linearized error dynamics around the nominal: δxt+1≈Atδxt+Btδut,At=∂f∂x|x‾t,u‾t,Bt=∂f∂u|x‾t,u‾t\delta x_{t+1} \approx A_{t}\,\delta x_{t} + B_{t}\,\delta u_{t}, \qquad A_{t} = \left.\frac{\partial f}{\partial x}\right|_{\bar{x}_{t},\bar{u}_{t}}, \quad B_{t} = \left.\frac{\partial f}{\partial u}\right|_{\bar{x}_{t},\bar{u}_{t}}

Quadratic cost (second-order Taylor expansion of ctc_{t} around nominal):

ct≈c‾t+[qtrt]T[δxδu]+12[δxδu]T[QtcostStTStRt][δxδu]c_{t} \approx \bar{c}_{t} + \begin{bmatrix}q_{t} \\ r_{t}\end{bmatrix}^{\!T} \begin{bmatrix}\delta x \\ \delta u\end{bmatrix} + \frac{1}{2}\begin{bmatrix}\delta x \\ \delta u\end{bmatrix}^{\!T} \begin{bmatrix} Q_t^{cost} & S_t^{\!T} \\ S_t & R_t\end{bmatrix}\begin{bmatrix}\delta x \\ \delta u\end{bmatrix}

Unlike LQR, the linear terms qtq_t and rtr_t are generally nonzero — the nominal is not yet optimal.

Backward Pass: Q-function

The Q-function for the deviation problem:

Q̃t(δx,δu)≈const+[QxQu]T[δxδu]+12[δxδu]T[QxxQxuQuxQuu][δxδu]\tilde{Q}_{t}(\delta x, \delta u) \approx \text{const} + \begin{bmatrix}Q_x \\ Q_u\end{bmatrix}^{\!T} \begin{bmatrix}\delta x \\ \delta u\end{bmatrix} + \frac{1}{2}\begin{bmatrix}\delta x \\ \delta u\end{bmatrix}^{\!T} \begin{bmatrix}Q_{xx} & Q_{xu} \\ Q_{ux} & Q_{uu}\end{bmatrix}\begin{bmatrix}\delta x \\ \delta u\end{bmatrix}

Backward recursion (initialize VT,x=cT,xV_{T,x} = c_{T,x}, VT,xx=cT,xxV_{T,xx} = c_{T,xx}):

Qx=qt+AtTVt+1,x,Qu=rt+BtTVt+1,xQ_x = q_t + A_{t}^T V_{t+1,x}, \qquad Q_u = r_t + B_{t}^T V_{t+1,x} Qxx=ct,xx+AtTVt+1,xxAt,Quu=ct,uu+BtTVt+1,xxBt,Qux=ct,ux+BtTVt+1,xxAtQ_{xx} = c_{t,xx} + A_{t}^T V_{t+1,xx} A_{t}, \quad Q_{uu} = c_{t,uu} + B_{t}^T V_{t+1,xx} B_{t}, \quad Q_{ux} = c_{t,ux} + B_{t}^T V_{t+1,xx} A_{t} Qxx=Qtcost+AtTVt+1,xxAt,Quu=Rt+BtTVt+1,xxBt,Qux=St+BtTVt+1,xxAtQ_{xx} = Q_t^{cost} + A_{t}^T V_{t+1,xx} A_{t}, \quad Q_{uu} = R_t + B_{t}^T V_{t+1,xx} B_{t}, \quad Q_{ux} = S_t + B_{t}^T V_{t+1,xx} A_{t}

Backward Pass: Optimal Gains

Minimize Q̃t\tilde{Q}_{t} over δu\delta u (holding δx\delta x fixed): ∂Q̃t∂δu=Qu+Quuδu+Quxδx=0\frac{\partial \tilde{Q}_{t}}{\partial \,\delta u} = Q_u + Q_{uu}\,\delta u + Q_{ux}\,\delta x = 0

⟹δut*=−Quu−1Qu⏟kt+(−Quu−1Qux)⏟Ktδxt\implies \delta u_{t}^* = \underbrace{-Q_{uu}^{-1} Q_u}_{k_{t}} + \underbrace{(-Q_{uu}^{-1} Q_{ux})}_{K_{t}}\,\delta x_{t}

Gain Expression Role
kt∈ℝmk_{t} \in \mathbb{R}^m −Quu−1Qu-Q_{uu}^{-1} Q_u Feedforward — optimization step
Kt∈ℝm×nK_{t} \in \mathbb{R}^{m \times n} −Quu−1Qux-Q_{uu}^{-1} Q_{ux} Feedback — state-dependent correction

Value function updated as: Vt,x=Qx+QxuktV_{t,x} = Q_x + Q_{xu} k_{t}, Vt,xx=Qxx+QxuKt\quad V_{t,xx} = Q_{xx} + Q_{xu} K_{t}

At the optimum Qu=rt+BtTVt+1,x=0Q_u = r_t + B_{t}^T V_{t+1,x} = 0, so kt=0k_{t} = 0 — identical to LQR.

Forward Pass

The Forward Pass Update

Starting from x̂0=x‾0\hat{x}_0 = \bar{x}_0, for t=0,1,…,T−1t = 0, 1, \dots, T-1:

ût=u‾t+αkt+Kt(x̂t−x‾t)⏟δxt\boxed{\hat{u}_{t} = \bar{u}_{t} + \alpha\, k_{t} + K_{t}\underbrace{(\hat{x}_{t} - \bar{x}_{t})}_{\delta x_{t}}} x̂t+1=f(x̂t,ût)←true nonlinear dynamics\hat{x}_{t+1} = f\!\left(\hat{x}_{t},\; \hat{u}_{t}\right) \qquad \leftarrow \text{true nonlinear dynamics}


Discussion: α\alpha multiplies ktk_{t} but not KtK_{t}. Shouldn’t we scale the entire update δut=α(kt+Ktδxt)\delta u_{t} = \alpha(k_{t} + K_{t} \delta x_{t})?

Why α\alpha Scales Only ktk_{t}

Both ktk_{t} and KtδxtK_{t} \delta x_{t} arise from minimizing Q̃t\tilde{Q}_{t} — both are part of the optimal solution to the linearized sub-problem. The distinction is open-loop vs. closed-loop:

Term Character Role
kt=−Quu−1Quk_{t} = -Q_{uu}^{-1} Q_u Open-loop (feedforward) Depends only on tt; the search direction
KtδxtK_{t} \delta x_{t} Closed-loop (feedback) Reacts to whatever δxt\delta x_{t} the nonlinear rollout produces

Connection to LQR: In LQR the full optimal policy is ut*=−Ktxtu_t^* = -K_t x_t — the feedback gain IS the solution, not a correction. iLQR adds the feedforward ktk_t only because Qu≠0Q_u \neq 0 off the optimum; at convergence kt→0k_t \to 0 and only the feedback remains.

Why α\alpha Scales Only ktk_{t}

Why α\alpha does not multiply KtδxtK_{t} \delta x_{t}:

α\alpha is a step size along the open-loop search direction ktk_{t}. The closed-loop term KtδxtK_{t} \delta x_{t} should always be applied at full strength — δxt\delta x_{t} is determined by the nonlinear dynamics, not by how aggressively we step. Scaling it by α\alpha would under-correct for nonlinear drift without any line-search benefit.

Special case — linearized forward pass: if δxt\delta x_{t} propagates through Atδxt+BtδutA_{t} \delta x_{t} + B_{t} \delta u_{t}, then δxt\delta x_{t} is also a Newton direction and one could scale the full pair α(kt+Ktδxt)\alpha(k_{t} + K_{t} \delta x_{t}). The nonlinear pass breaks this symmetry, leaving ktk_{t} as the only clean search direction.

Thought Experiment

Forward pass: ût=u‾t+αkt+Kt(x̂t−x‾t)\hat{u}_{t} = \bar{u}_{t} + \alpha\, k_{t} + K_{t}(\hat{x}_{t} - \bar{x}_{t})

α=0\alpha = 0: ût=u‾t+Kt(x̂t−x‾t)\quad \hat{u}_{t} = \bar{u}_{t} + K_{t}(\hat{x}_{t} - \bar{x}_{t})

  • Feedforward ktk_{t} is switched off — no optimization progress
  • KtK_{t} corrects for nonlinear drift, keeping the trajectory near nominal
  • Cost stays approximately the same

α=1\alpha = 1: Full Newton step on the open-loop component.

  • KtδxtK_{t} \delta x_{t} is applied at full strength throughout — its magnitude is set by the nonlinear dynamics, not by α\alpha

At t=0t = 0: x̂0=x‾0\hat{x}_0 = \bar{x}_0 always, so δx0=0\delta x_0 = 0.

  • The feedback plays no role at the first step regardless of α\alpha
  • Only αk0\alpha k_0 changes the control there
  • Feedback activates only as deviations accumulate along the horizon

Nonlinear vs. Linearized

Two Choices for the Forward Pass

Nonlinear (standard iLQR) Linearized
State propagation x̂t+1=f(x̂t,ût)\hat{x}_{t+1} = f(\hat{x}_{t}, \hat{u}_{t}) δxt+1=Atδxt+Btδut\delta x_{t+1} = A_{t} \delta x_{t} + B_{t} \delta u_{t}
Trajectory feasible? Yes — on true dynamics No — satisfies only linear model
Is δxt\delta x_{t} a Newton direction? No — nonlinear residuals corrupt it Yes
α\alpha appropriately scales Only ktk_{t} Full α(kt+Ktδxt)\alpha(k_{t} + K_{t} \delta x_{t})
Best suited for Highly nonlinear systems Near-optimal, mildly nonlinear

Near convergence, linearization error →0\to 0 and the two approaches become equivalent. The difference matters most in the early iterations.

Line Search

Backtracking Line Search

Goal: find α∈(0,1]\alpha \in (0, 1] such that the updated trajectory strictly reduces cost.

Algorithm:

  1. Set α=1\alpha = 1
  2. Run nonlinear forward pass with current α\alpha; evaluate J(û)J(\hat{u})
  3. If J(û)<J(u‾)J(\hat{u}) < J(\bar{u}): accept, update x‾←x̂\bar{x} \leftarrow \hat{x}, u‾←û\bar{u} \leftarrow \hat{u}
  4. Else: α←α/2\alpha \leftarrow \alpha/2, repeat from step 2
  5. After 50 halvings: if α<2×10−6\alpha < 2\times 10^{-6}, declare convergence
alpha = 1.0
for _ in range(50):
    for t in range(T):
        delta_u = K[t] @ (x_new[t] - x_nom[t]) + alpha * k_ff[t]
        u_new[t] = u_nom[t] + delta_u
        x_new[t+1] = f(x_new[t], u_new[t])   # nonlinear step
    if new_cost < current_cost:
        break
    alpha *= 0.5

Predicted Cost Improvement

The backward pass implicitly computes the predicted improvement:

ΔJ=∑k=0T−1(QuTkt+12ktTQuukt)=−12∑k=0T−1ktTQuukt≤0\Delta J = \sum_{k=0}^{T-1}\left(Q_u^T k_{t} + \frac{1}{2} k_{t}^T Q_{uu} k_{t}\right) = -\frac{1}{2}\sum_{k=0}^{T-1} k_{t}^T Q_{uu} k_{t} \;\leq\; 0

This motivates a stronger Armijo condition:

J(û)<J(u‾)−γα|ΔJ|,γ∈(0,1)J(\hat{u}) < J(\bar{u}) - \gamma\,\alpha\,|\Delta J|, \qquad \gamma \in (0,1)

Actual decrease must be at least a fraction γ\gamma of the quadratic model’s prediction.

At convergence: kt→0k_{t} \to 0 (because Qu→0Q_u \to 0), so ΔJ→0\Delta J \to 0 and α=1\alpha = 1 is accepted immediately on the first try — the line search costs nothing.

Exercise: Optimality at Convergence

At convergence, kt=−Quu−1Qu→0k_{t} = -Q_{uu}^{-1} Q_u \to 0, which means:

Qu=rt+BtTVt+1,x=0for all tQ_u = r_t + B_{t}^T V_{t+1,x} = 0 \quad \text{for all } t

Question: Write out what this says in terms of the original cost and dynamics. What classical optimality condition does it correspond to?

Answer: This is the discrete-time Pontryagin Maximum Principle stationarity condition:

∂ct∂ut+BtTλt+1=0,λt+1=Vt+1,x\frac{\partial c_{t}}{\partial u_{t}} + B_{t}^T \lambda_{t+1} = 0, \qquad \lambda_{t+1} = V_{t+1,x}

where λt\lambda_{t} is the costate (adjoint variable). iLQR converges precisely when the KKT conditions are satisfied.

Full Algorithm

iLQR: Complete Algorithm

iLQR

Input: initial controls u‾0:T−1\bar{u}_{0:T-1}, dynamics ftf_{t}, costs ctc_{t}, cTc_{T}

1. Rollout: simulate x‾t+1=ft(x‾t,u‾t)\bar{x}_{t+1} = f_{t}(\bar{x}_{t}, \bar{u}_{t}) to obtain the nominal trajectory.

2. Repeat until convergence:

2a. Backward pass — init VT,x=cT,xV_{T,x} = c_{T,x}, VT,xx=cT,xxV_{T,xx} = c_{T,xx}; for t=T−1,…,0t = T-1, \dots, 0:

  • Compute Jacobians At,BtA_{t}, B_{t} and cost derivatives at (x‾t,u‾t)(\bar{x}_{t}, \bar{u}_{t})
  • Compute Qx,Qu,Qxx,Quu,QuxQ_x, Q_u, Q_{xx}, Q_{uu}, Q_{ux} via the recursion
  • Gains: kt=−Quu−1Quk_{t} = -Q_{uu}^{-1} Q_u, Kt=−Quu−1Qux\quad K_{t} = -Q_{uu}^{-1} Q_{ux}
  • Update: Vt,x=Qx+QxuktV_{t,x} = Q_x + Q_{xu} k_{t}, Vt,xx=Qxx+QxuKt\quad V_{t,xx} = Q_{xx} + Q_{xu} K_{t}

2b. Forward pass — from x̂0=x‾0\hat{x}_0 = \bar{x}_0, simulate with ût=u‾t+αkt+Kt(x̂t−x‾t)\hat{u}_{t} = \bar{u}_{t} + \alpha k_{t} + K_{t}(\hat{x}_{t} - \bar{x}_{t})

2c. Line search — if J(û)<J(u‾)J(\hat{u}) < J(\bar{u}): accept; else α←α/2\alpha \leftarrow \alpha/2, repeat 2b

2d. Update — x‾t←x̂t\bar{x}_{t} \leftarrow \hat{x}_{t}, u‾t←ût\;\bar{u}_{t} \leftarrow \hat{u}_{t}

The backward pass is structurally identical to LQR applied to the deviation problem. The only additions: (1) feedforward gains ktk_{t} from nonzero cost gradients; (2) re-linearize each iteration.

Extensions

Beyond iLQR

Differential Dynamic Programming (DDP):

iLQR uses only first-order Jacobians of ftf_{t}. DDP adds second-order terms:

Qxx+=∑i(Vt+1,x)(i)ft,xx(i),Quu+=⋯,Qux+=⋯Q_{xx} \mathrel{+}= \sum_i (V_{t+1,x})^{(i)} f_{t,xx}^{(i)}, \quad Q_{uu} \mathrel{+}= \cdots, \quad Q_{ux} \mathrel{+}= \cdots

  • Terms scale as 𝒪(∥Vt+1,x∥)\mathcal{O}(\|V_{t+1,x}\|) — vanish near the optimum where Vt+1,x→0V_{t+1,x} \to 0
  • Better convergence rate far from the optimum; expensive and rarely used in practice
  • iLQR is sometimes called Gauss-Newton DDP

Summary

Forward pass update: ût=u‾t+αkt+Kt(x̂t−x‾t),x̂t+1=f(x̂t,ût)\hat{u}_{t} = \bar{u}_{t} + \alpha\, k_{t} + K_{t}(\hat{x}_{t} - \bar{x}_{t}), \qquad \hat{x}_{t+1} = f(\hat{x}_{t}, \hat{u}_{t})

  • α\alpha scales only ktk_{t} — the Newton search direction in uu
  • KtδxtK_{t} \delta x_{t} corrects for nonlinear drift; it is not a search direction unless dynamics are linear

Line search: halve α\alpha until JJ decreases; kt→0k_{t} \to 0 signals convergence

Convergence ↔︎ optimality: Qu=0Q_u = 0 is the discrete-time PMP stationarity condition
